# Project: Bach Landing Page

Scroll landing page for a contracting firm. More focused on interior design. Minimalist, clean, whitespace-focused, no animations.

## Tech Stack

Next.js (App Router) + Tailwind CSS

## Colors

| Token        | Hex     |
| ------------ | ------- |
| Primary Blue | #47709a |
| Hover Blue   | #3a5d82 |
| Silver/Grey  | #c9cbce |
| Dark Text    | #1a1a1a |
| White        | #ffffff |
| Light BG     | #f8f9fa |

## Typography

- **Headings:** Montserrat (700 bold, 500 medium)
- **Body:** Inter (400 regular)
- **Line height:** Headings 1.2, Body 1.6
- **Scale:** h1 48-56px, h2 36-40px, h3 24-28px, body 16-18px, small 14px

## Spacing (Tailwind)

- Elements: 4px, 8px, 12px, 16px
- Components: 24px, 32px
- Sections: 64px, 80px, 96px

## Responsive

- Mobile-first
- Primary breakpoint: md (768px)
- Container: max-w-6xl mx-auto px-4 md:px-6 lg:px-8

## Border Radius

Buttons: rounded-md | Cards: rounded-lg | Inputs: rounded
